# Sistema Contable

Sistema de gestión contable desarrollado con React y Vite.

## Características

- Gestión de ventas y compras
- Control de efectivo y banco
- Cuentas por cobrar y pagar
- Gestión de servicios mensuales
- Egresos recurrentes
- Gestión de personal y nómina
- Informes y reportes

## Tecnologías

- React 18
- Vite
- Bootstrap 5
- React Router
- Supabase
- XLSX

## Instalación

1. Clonar el repositorio
2. Instalar dependencias:
```bash
npm install
```
3. Iniciar servidor de desarrollo:
```bash
npm run dev
```

## Estructura del Proyecto

```
sistema-contable/
├── src/
│   ├── components/         # Componentes React
│   ├── pages/             # Páginas/Rutas
│   ├── utils/             # Utilidades y helpers
│   ├── styles/            # Estilos CSS
│   ├── App.jsx           # Componente principal
│   └── main.jsx          # Punto de entrada
├── public/               # Archivos estáticos
├── index.html           # HTML principal
└── package.json         # Dependencias y scripts
```
